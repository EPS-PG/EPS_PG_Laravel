# EPSPG Laravel

Official Laravel package for integrating **EPSPG (Easy Payment System)** Payment Gateway.

## Features
- Sandbox & Live mode support  
- Payment initialization  
- Callback & IPN handling  
- Simple Laravel-friendly API  

## Requirements
- PHP 8.0+
- Laravel 8 / 9 / 10
